Hello there! This is the first blog installment in the "Head In The Clouds Blog". In this blog, we will be discussing a new and exciting programming standard for the web called Web Assembly (WASM). This model allows code from languages like Rust, C++, Go, and more to be executed alongside JavaScript in a webpage! To see our first example of such a program supported by the standard, let's start with the canonical example one uses when exploring any new programming paradigm: the Hello World program.
